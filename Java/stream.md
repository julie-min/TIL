# Stream API
- Java8에서 추가된 기능
- 컬렉션 데이터를 선언형으로 쉽게 처리
- 루프문을 사용하지 않아도 되며, 중첩을 사용할 필요 없음.
- 멀티 쓰레드 구현 없이 사용

---

# Stream 특징
- 내부 반복을 통한 작업을 수행한다.
- 단 한번만 사용이 가능하다.
- 원본 데이터를 변경하지 않는다.
- 필터-맵 기반의 API를 사용하여 지연 연산을 통해 성능을 최적화 시킨다.
- 손쉬운 병렬처리가 가능하다.


# Stream API의 동작 흐름
1. 스트림을 생성한다.
2. 스트림의 중개연산 (변환) => 필터
3. 스트림의 최종연산 (사용) => 맵
