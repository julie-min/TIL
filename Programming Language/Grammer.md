# 프로그래밍 언어론
## 4. 구문론과 의미론

## 프로그램
- 프로그램의 구조
- 어휘: 토큰 - 문자의 모임, 최소한의 의미를 갖는 단어
- 구문: 프로그램을 작성하는 규칙
- 즉, 토큰을 모아 프로그램을 구성한다.

## 구문론 Syntax
- 문장이 구성되는 방식에 대해 연구
- 프로그램의 표면적인 구조를 정의
- 프로그램 작성시 어떤 형태로 작성해야 하는지를 기술


## 의미론 Semantics
- 문장이 나타내는 의미에 대해 연구
- 프로그램의 내용적인 효과를 정의
- 프로그램 실행시 어떤 일이 일어나는지 그 의미를 기술
